📋 Langkah Setup Supabase (Gratis)
1. Buka supabase.com → Sign up gratis
2. Buat project baru → catat Project URL dan anon public key (dari Settings → API)
3. Buka SQL Editor → jalankan query berikut untuk buat tabel: 
CREATE TABLE IF NOT EXISTS app_state (
  id TEXT PRIMARY KEY,
  data JSONB,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Enable Row Level Security (opsional tapi disarankan)
ALTER TABLE app_state ENABLE ROW LEVEL SECURITY;

-- Policy: izinkan semua operasi dengan anon key
CREATE POLICY "Allow all" ON app_state
  FOR ALL USING (true) WITH CHECK (true);
